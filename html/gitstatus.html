Git status

dictionaryDefinición

El comando git status muestra el estado actual del repositorio. Permite ver qué archivos han sido modificados, cuáles están listos para ser confirmados (staged) y cuáles no están siendo rastreados por Git (untracked). Es una de las herramientas más útiles para saber en qué punto se encuentra tu proyecto antes de hacer un commit o un push.

keyboard_command_keyComandos

  • git statusarrow_right

    Muestra el estado actual del repositorio.

  • git status -sarrow_right

    Muestra el estado en formato corto, ideal para una vista más compacta.

  • git status -barrow_right

    Incluye información sobre la rama actual y su relación con la rama remota.

terminal Ejecutar

$ git status
