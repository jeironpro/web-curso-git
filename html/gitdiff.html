Git diff

dictionaryDefinición

El comando git diff muestra las diferencias entre los cambios realizados en tu proyecto. Permite comparar lo que ha cambiado en los archivos antes de preparar (staging) o entre commits específicos. Es ideal para revisar el código antes de confirmar los cambios con git commit.

keyboard_command_keyComandos

  • git diffarrow_right

    Muestra las diferencias entre archivos en tu directorio de trabajo y el staging area, o entre commits.

  • git diff --cachedarrow_right

    Muestra las diferencias entre el área de trabajo y el área de preparación (staging area).

  • git diff --stagedarrow_right

    Compara los cambios ya añadidos al área de preparación con el último commit.

  • git diff HEADarrow_right

    Compara el estado actual del proyecto con el último commit.

  • git diff <commit1> <commit2>arrow_right

    Muestra las diferencias entre dos commits específicos.

terminal Ejecutar

echo "console.log('Las diferencias en Git');" >> main.js
$ git diff
$ git status
$ git add main.js
$ git commit -m "Modificando una nueva vez main.js"
