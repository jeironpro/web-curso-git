Git clone

dictionaryDefinición

El comando git clone se utiliza para crear una copia exacta de un repositorio remoto en tu máquina local. Este comando descarga todo el historial de versiones, ramas y archivos del proyecto, permitiéndote trabajar de forma local con una réplica completa del repositorio original.

keyboard_command_keyComandos

  • git clone <url>arrow_right

    Copia un repositorio remoto completo a tu máquina local.

  • git clone <url> <directorio>arrow_right

    Clona el repositorio en un directorio específico, en lugar de usar el nombre por defecto.

  • git clone --branch <nombre_rama> <url>arrow_right

    Clona únicamente una rama específica del repositorio remoto.

  • git clone --depth <n> <url>arrow_right

    Realiza una clonación superficial con solo los últimos numeros de commits (ideal para ahorrar espacio o tiempo).

  • git clone --bare <url>arrow_right

    Crea un clon sin área de trabajo (solo los metadatos del repositorio, útil para servidores).
