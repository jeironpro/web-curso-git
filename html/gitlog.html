Git log

dictionaryDefinición

El comando git log se utiliza para mostrar el historial de commits de un repositorio. Permite ver información detallada de cada commit, como el identificador único (hash), autor, fecha y mensaje. Es útil para revisar cambios pasados y entender la evolución del proyecto.

keyboard_command_keyComandos

  • git logarrow_right

    Muestra el historial completo de commits en la rama actual.

  • git log --onelinearrow_right

    Muestra cada commit en una sola línea, mostrando solo el hash abreviado y el mensaje.

  • git log -parrow_right

    Muestra los cambios de cada commit (diferencias) además del historial.

  • git log --statarrow_right

    Muestra un resumen de archivos modificados y cantidad de líneas añadidas o eliminadas por commit.

terminal Ejecutar

$ git log
