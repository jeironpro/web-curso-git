Git stash

dictionaryDefinición

El comando git stash se utiliza para guardar temporalmente los cambios en el área de trabajo y el área de preparación sin hacer un commit. Es ideal cuando necesitas cambiar de rama o realizar otra tarea, pero aún no quieres confirmar tus modificaciones actuales. Los cambios se guardan en una pila llamada stash stack, desde donde puedes recuperarlos más tarde.

keyboard_command_keyComandos

  • git stasharrow_right

    Guarda temporalmente los cambios que aún no has confirmado (staging area o working directory) y limpia tu directorio de trabajo.

  • git stash listarrow_right

    Muestra todas las entradas guardadas en la pila de stashes.

  • git stash poparrow_right

    Restaura los cambios del stash más reciente y lo elimina de la pila.

  • git stash apply stash@{n}arrow_right

    Aplica un stash específico sin eliminarlo de la pila.

  • git stash drop stash@{n}arrow_right

    Elimina una entrada concreta del stash.

  • git stash cleararrow_right

    Elimina todos los stashes almacenados.

terminal Ejecutar

$ echo "console.log('Un stash');" >> main.js
$ git status
$ git stash
$ git status
$ git stash list
$ git stash pop
$ git add main.js
$ git status
$ git commit -m "Modificando main.js"
