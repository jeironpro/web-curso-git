Git checkout

dictionaryDefinición

El comando git checkout se utiliza para cambiar entre ramas o restaurar archivos en un repositorio. Permite mover HEAD a otra rama o commit, o recuperar versiones anteriores de archivos específicos. Aunque git switch y git restore son comandos más modernos, git checkout sigue siendo ampliamente usado.

keyboard_command_keyComandos

  • git checkout <nombre_rama>arrow_right

    Cambia a la rama indicada y actualiza el directorio de trabajo con su contenido.

  • git checkout -b <nueva_rama>arrow_right

    Crea una nueva rama y se cambia automáticamente a ella.

  • git checkout <hash> -- <archivo>arrow_right

    Restaura un archivo específico desde un commit determinado.

  • git checkout HEAD -- <archivo>arrow_right

    Restaura un archivo al estado del último commit en HEAD.

terminal Ejecutar

$ git branch
$ git checkout backend
