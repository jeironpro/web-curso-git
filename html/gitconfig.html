Git config

dictionaryDefinición

El comando git config se utiliza para ver y establecer las configuraciones de Git. Permite definir información del usuario, preferencias de formato, colores, editores predeterminados y opciones globales o locales. Las configuraciones pueden aplicarse a nivel local (solo el repositorio actual), global (para todos los repositorios del usuario) o de sistema (para todos los usuarios del equipo).

keyboard_command_keyComandos

  • git config --global user.name "<nombre>"arrow_right

    Establece el nombre del usuario para todos los commits realizados en el sistema.

  • git config --global user.email "<correo>"arrow_right

    Configura el correo electrónico del usuario globalmente.

  • git config --listarrow_right

    Muestra todas las configuraciones actuales de Git (locales, globales y de sistema).

  • git config --edit --globalarrow_right

    Abre el archivo global de configuración en el editor por defecto para editarlo manualmente.

  • git config core.editor "code --wait"arrow_right

    Define Visual Studio Code como editor predeterminado para Git.

terminal Ejecutar

$ git config --global user.name "jeironpro"
$ git config --global user.email "jeironprogrammer@gmail.com"
$ git config --list

insights Resultado

user.name=jeironpro
user.email=jeironprogrammer@gmail.com
