keyConexión SSH entre Git y GitHub

  • En tu terminal:
    • Genera una nueva clave SSH:
      • ssh-keygen -t ed25519 -C "your_email@example.com"
    • > Enter a file in which to save the key (/home/YOU/.ssh/id_ALGORITHM): [Presiona enter]
    • > Enter passphrase (empty for no passphrase): [Presiona enter]
    • > Enter same passphrase again: [Presiona enter]
    • Inicia el agente SSH:
      • eval "$(ssh-agent -s)"
    • Agrega la clave al agente:
      • ssh-add ~/.ssh/id_ed25519
    • Muestra la clave pública y cópiala:
      • cat ~/.ssh/id_ed25519.pub
  • En GitHub:
    • Abre GitHub y ve a Settings.
    • Selecciona SSH and GPG Keys.
    • Haz clic en New SSH Key.
    • En Title, escribe un nombre para la clave.
    • En Key Type, selecciona Authentication Key.
    • Pega el contenido copiado del archivo id_ed25519.pub.
    • Haz clic en Add SSH Key.
