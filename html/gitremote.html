Git remote

dictionaryDefinición

El comando git remote se utiliza para administrar las conexiones entre el repositorio local y los repositorios remotos. Permite agregar, eliminar, renombrar o listar los repositorios remotos asociados, facilitando la colaboración y la sincronización de cambios en proyectos compartidos.

keyboard_command_keyComandos

  • git remotearrow_right

    Muestra los repositorios remotos asociados a tu proyecto Git local.

  • git remote -varrow_right

    Muestra una lista de los repositorios remotos configurados junto con sus URLs de lectura y escritura.

  • git remote add <nombre> <url>arrow_right

    Agrega un nuevo repositorio remoto con el nombre y la URL especificados.

  • git remote remove <nombre>arrow_right

    Elimina un repositorio remoto del listado de conexiones.

  • git remote rename <antiguo> <nuevo>arrow_right

    Cambia el nombre de un repositorio remoto existente.

  • git remote set-url <nombre> <nueva_url>arrow_right

    Modifica la URL de un remoto ya configurado (útil si la dirección del repositorio cambió).

terminal Ejecutar

$ git remote add origin https://github.com/usuario/mi-proyecto.git
$ git remote -v
