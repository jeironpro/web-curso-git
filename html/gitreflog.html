Git reflog

dictionaryDefinición

El comando git reflog se utiliza para ver el historial de referencias de las ramas y HEAD en el repositorio. Permite rastrear todos los movimientos de HEAD, incluyendo commits, resets, checkouts y merges, aunque no estén en el historial normal. Es útil para recuperar commits perdidos o revertir cambios recientes.

keyboard_command_keyComandos

  • git reflogarrow_right

    Muestra un registro de todos los movimientos de HEAD en tu repositorio, incluyendo commits, resets, merges y cambios de rama.

  • git reflog show <rama>arrow_right

    Muestra el historial de una rama específica.

  • git reset --hard <hash>arrow_right

    Permite regresar a un estado previo usando un hash obtenido de reflog.

  • git reflog expire --expire=now --allarrow_right

    Limpia los registros antiguos del reflog según la política indicada.

terminal Ejecutar

$ git reflog
