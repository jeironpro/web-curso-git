Git show

dictionaryDefinición

El comando git show se utiliza para mostrar información detallada de un commit específico. Permite ver el mensaje del commit, los cambios realizados, y las diferencias (diff) en los archivos modificados. Es útil para inspeccionar un commit individual sin navegar todo el historial.

keyboard_command_keyComandos

  • git show <hash_del_commit>arrow_right

    Muestra los detalles de un commit específico identificado por su hash.

  • git show --name-only <hash_del_commit>arrow_right

    Muestra únicamente los nombres de los archivos modificados en ese commit.

  • git show --stat <hash_del_commit>arrow_right

    Muestra un resumen de los archivos modificados y líneas añadidas o eliminadas en el commit.

  • git show --patch <hash_del_commit>arrow_right

    Muestra los cambios completos (diff) introducidos por el commit.

terminal Ejecutar

$ git log
$ git show b572aa5b1959526fe7d8f0361a81e97e90d53082
