Git fetch

dictionaryDefinición

El comando git fetch se utiliza para descargar los cambios de un repositorio remoto sin fusionarlos automáticamente con la rama local. Permite actualizar referencias remotas y obtener commits recientes, tags y ramas sin modificar tu directorio de trabajo. Es útil para inspeccionar cambios remotos antes de integrarlos.

keyboard_command_keyComandos

  • git fetcharrow_right

    Descarga todos los cambios del repositorio remoto configurado (por defecto 'origin').

  • git fetch origin mainarrow_right

    Descarga los cambios de la rama main del remoto origin sin fusionarlos.

  • git fetch --allarrow_right

    Descarga los cambios de todos los remotos configurados.

  • git fetch --prunearrow_right

    Elimina referencias remotas que ya no existen en el servidor al realizar la descarga.
