Git branch

dictionaryDefinición

El comando git branch se utiliza para listar, crear o eliminar ramas en un repositorio de Git. Las ramas permiten desarrollar funcionalidades de manera aislada sin afectar la rama principal. Es fundamental para gestionar el flujo de trabajo y la colaboración en proyectos.

keyboard_command_keyComandos

  • git brancharrow_right

    Muestra todas las ramas locales del repositorio y la rama actual marcada con un asterisco.

  • git branch <nombre_rama>arrow_right

    Crea una nueva rama local con el nombre indicado.

  • git branch -m <nombre_antiguo> <nombre_nuevo>arrow_right

    Renombra una rama local.

  • git branch -d <nombre_rama>arrow_right

    Elimina una rama local que ya ha sido fusionada.

terminal Ejecutar

$ git branch
$ git branch frontend
$ git branch backend
$ git branch
$ git branch -m frontend frontend_react
$ git branch
$ git branch -d frontend_react
$ git branch
