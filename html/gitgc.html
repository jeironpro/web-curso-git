Git gc

dictionaryDefinición

El comando git gc (garbage collection) se utiliza para optimizar el repositorio Git. Combina objetos sueltos, comprime datos y limpia referencias innecesarias, mejorando el rendimiento y reduciendo el tamaño del repositorio. Es recomendable ejecutarlo periódicamente, especialmente en repositorios grandes.

keyboard_command_keyComandos

  • git gcarrow_right

    Realiza una recolección de basura estándar optimizando el repositorio.

  • git gc --aggressivearrow_right

    Ejecuta una recolección más exhaustiva y agresiva, optimizando aún más el repositorio a costa de más tiempo de ejecución.

  • git gc --prune=nowarrow_right

    Elimina inmediatamente objetos inalcanzables sin esperar el tiempo de retención por defecto.

terminal Ejecutar

$ git branch
$ git checkout main
$ git push
$ git gc
