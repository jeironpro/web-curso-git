Git submodule

dictionaryDefinición

El comando git submodule se utiliza para agregar, inicializar y administrar repositorios Git dentro de otro repositorio principal. Los submódulos permiten incluir proyectos externos como dependencias manteniendo sus propios historiales de commits. Es útil para organizar dependencias de proyectos o librerías compartidas.

keyboard_command_keyComandos

  • git submodule add <repositorio_url> <ruta>arrow_right

    Agrega un submódulo en la ruta especificada apuntando al repositorio indicado.

  • git submodule initarrow_right

    Inicializa la configuración de los submódulos registrados en .gitmodules.

  • git submodule updatearrow_right

    Clona y actualiza los submódulos a la revisión registrada en el repositorio principal.

  • git submodule statusarrow_right

    Muestra el estado de los submódulos, indicando commits y si están actualizados.
