Git blame

dictionaryDefinición

El comando git blame muestra quién modificó cada línea de un archivo, junto con el hash del commit, la fecha y el autor. Es muy útil para investigar el historial de cambios y comprender cuándo y por qué se introdujo cierta modificación.

keyboard_command_keyComandos

  • git blame <archivo>arrow_right

    Muestra línea por línea quién modificó el archivo por última vez.

  • git blame -L 10,30 <archivo>arrow_right

    Muestra la información de las líneas comprendidas entre la 10 y la 30.

  • git blame --date short <archivo>arrow_right

    Usa un formato de fecha corto (YYYY-MM-DD) en los resultados.

  • git blame --show-email <archivo>arrow_right

    Muestra el correo electrónico del autor junto al nombre.

terminal Ejecutar

$ git blame main.js
