Git archive

dictionaryDefinición

El comando git archive se utiliza para crear un archivo comprimido (por ejemplo, .zip o .tar) del contenido de un commit o rama. Permite exportar el proyecto sin incluir el historial de Git, ideal para distribuciones o respaldos de código.

keyboard_command_keyComandos

  • git archive --format=zip HEAD -o proyecto.ziparrow_right

    Crea un archivo zip del commit actual (HEAD) y lo guarda como proyecto.zip.

  • git archive --format=tar main | gzip > proyecto.tar.gzarrow_right

    Crea un archivo tar.gz del contenido de la rama main.

  • git archive --prefix=proyecto/ HEAD | tar -cvf proyecto.tar -arrow_right

    Crea un tar con un prefijo de carpeta dentro del archivo.

  • git archive --output=backup.zip HEADarrow_right

    Exporta el commit actual a backup.zip usando la opción de salida.

terminal Ejecutar

$ git archive --format=zip main -o mi-proyecto.zip
$ git status
$ unzip mi-proyecto.zip -d mi-proyecto-copy
$ git status
