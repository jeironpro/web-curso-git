Git push

dictionaryDefinición

El comando git push se utiliza para enviar los commits de tu repositorio local al remoto. Actualiza la rama remota con tus cambios locales, permitiendo que otros colaboren y vean tus modificaciones. Es el paso final para sincronizar tu trabajo con el repositorio compartido.

keyboard_command_keyComandos

  • git pusharrow_right

    Envía los commits de la rama actual al remoto configurado por defecto (origin).

  • git push origin mainarrow_right

    Envía los commits de la rama main al remoto origin.

  • git push -u origin mainarrow_right

    Establece la rama remota main como upstream y envía los cambios.

  • git push --forcearrow_right

    Forza la actualización del remoto, sobrescribiendo cambios si es necesario. Usar con precaución.

terminal Ejecutar

$ git push
$ git push --set-upstream origin main
