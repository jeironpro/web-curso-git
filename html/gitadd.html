Git add

dictionaryDefinición

El comando git add se utiliza para preparar archivos antes de confirmarlos en un commit. Mueve los cambios del área de trabajo (working directory) al área de preparación (staging area), donde Git registra qué se incluirá en el siguiente commit. En otras palabras, git add le dice a Git: “estos cambios están listos para guardarse”.

keyboard_command_keyComandos

  • git add <archivo>arrow_right

    Agrega el archivos al área de preparación (staging area).

  • git add .arrow_right

    Agrega todos los archivos nuevos o modificados al área de preparación (staging area).

  • git add -Aarrow_right

    Agrega todos los cambios (incluidos archivos eliminados).

  • git add -parrow_right

    Permite agregar solo fragmentos específicos de los cambios en un archivo.

terminal Ejecutar

$ echo "console.log('Hola Git');" > app.js
$ git status
$ git add app.js
$ git status
