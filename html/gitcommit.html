Git commit

dictionaryDefinición

El comando git commit se utiliza para guardar los cambios en el historial del repositorio.Crea un nuevo commit que registra los cambios añadidos al área de preparación (staging area) y permite describirlos mediante un mensaje. Es un paso esencial para mantener un historial claro y comprensible de tu proyecto.

keyboard_command_keyComandos

  • git commit -m "mensaje"arrow_right

    Crea un commit con todos los cambios que estén en el staging area.

  • git commit -a -m "mensaje"arrow_right

    Hace commit de todos los archivos modificados y rastreados, saltándose git add.

  • git commit --amendarrow_right

    Modifica el último commit, ya sea para cambiar el mensaje o agregar cambios adicionales.

  • git commit -varrow_right

    Muestra los cambios que serán incluidos en el commit dentro del editor de mensaje.

terminal Ejecutar

$ git status
$ git commit -m "Mi primer commit"
