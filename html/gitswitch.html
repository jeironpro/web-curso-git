Git switch

dictionaryDefinición

El comando git switch se utiliza para cambiar de rama dentro de un repositorio de Git. Es una alternativa moderna a git checkout enfocada únicamente en la gestión de ramas, haciendo el flujo más claro y seguro al cambiar de contexto de trabajo.

keyboard_command_keyComandos

  • git switch <nombre_rama>arrow_right

    Cambia a la rama indicada y actualiza el directorio de trabajo con su contenido.

  • git switch -c <nueva_rama>arrow_right

    Crea una nueva rama y se cambia automáticamente a ella.

  • git switch -C <nueva_rama>arrow_right

    Fuerza la creación de una rama nueva, sobrescribiendo si ya existía.

  • git switch -arrow_right

    Cambia a la rama anterior en la que estabas trabajando.

terminal Ejecutar

$ git branch
$ git switch main
$ git branch
