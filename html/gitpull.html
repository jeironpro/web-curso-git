Git pull

dictionaryDefinición

El comando git pull se utiliza para actualizar la rama actual con los cambios del repositorio remoto. Combina automáticamente git fetch (descargar cambios) y git merge (fusionarlos). Es útil para mantener tu repositorio local sincronizado con el remoto.

keyboard_command_keyComandos

  • git pullarrow_right

    Actualiza la rama actual desde su remoto configurado (por defecto origin).

  • git pull origin mainarrow_right

    Trae y fusiona los cambios de la rama main del remoto origin.

  • git pull --rebasearrow_right

    Actualiza la rama actual aplicando tus commits locales sobre los cambios remotos (rebase).

  • git pull --ff-onlyarrow_right

    Actualiza solo si se puede hacer un avance rápido; evita fusiones automáticas.

terminal Ejecutar

$ git pull origin main
