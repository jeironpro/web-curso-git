Git clean

dictionaryDefinición

El comando git clean se utiliza para eliminar archivos y directorios no rastreados del área de trabajo. Es especialmente útil cuando has creado archivos temporales, binarios o pruebas que no deseas agregar al repositorio. Advertencia: este comando borra archivos del disco, por lo que debe usarse con precaución.

keyboard_command_keyComandos

  • git clean -narrow_right

    Muestra una vista previa de los archivos que serían eliminados sin borrarlos realmente.

  • git clean -farrow_right

    Elimina definitivamente los archivos no rastreados.

  • git clean -dfarrow_right

    Elimina archivos y también carpetas no rastreadas.

  • git clean -xarrow_right

    Elimina también los archivos ignorados por .gitignore.

terminal Ejecutar

$ echo "Git es impresionante" > hola_git.txt
$ git status
$ git clean -f hola_git.txt
$ git status
