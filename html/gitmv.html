Git mv

dictionaryDefinición

El comando git mv se utiliza para mover o renombrar archivos y directorios dentro de un repositorio de Git. Combina de forma automática los pasos de mover el archivo, eliminar el anterior y agregar el nuevo al área de preparación. Es preferible usar git mv en lugar de comandos del sistema (como mv o rename), ya que Git mantiene el historial y detecta correctamente el cambio.

keyboard_command_keyComandos

  • git mv <archivo> <nuevo_nombre_archivo>arrow_right

    Renombra el archivo dentro del repositorio.

  • git mv <archivo> <directorio>/arrow_right

    Mueve el archivo a otro directorio dentro del proyecto.

  • git mv -f <archivo> <nuevo_nombre>arrow_right

    Forza el renombrado o movimiento si el archivo de destino ya existe.

terminal Ejecutar

$ ls
$ git mv app.js main.js
$ git status
