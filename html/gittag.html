Git tag

dictionaryDefinición

El comando git tag se utiliza para crear, listar o eliminar etiquetas (tags) en un repositorio de Git. Las etiquetas son referencias fijas a commits específicos y suelen usarse para marcar versiones o hitos importantes. Es útil para versionar y distribuir releases de manera clara.

keyboard_command_keyComandos

  • git tagarrow_right

    Muestra todas las etiquetas (tags) que existen en el repositorio.

  • git tag v1.0arrow_right

    Crea una etiqueta ligera llamada v1.0 en el commit actual.

  • git tag -a v1.0 -m "Versión 1.0"arrow_right

    Crea una etiqueta anotada con mensaje explicativo.

  • git tag -d v1.0arrow_right

    Elimina una etiqueta localmente.

terminal Ejecutar

$ git tag v1.0 -m "Versión 1.0"
$ git tag
