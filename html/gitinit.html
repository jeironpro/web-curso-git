Git init

dictionaryDefinición

El comando git init se utiliza para inicializar un nuevo repositorio de Git en un directorio. Esto crea una carpeta oculta .git que contiene toda la información necesaria para el control de versiones del proyecto.

keyboard_command_keyComandos

  • git initarrow_right

    Inicializa un repositorio Git vacío en el directorio actual.

  • git init --barearrow_right

    Crea un repositorio sin área de trabajo (ideal para servidores remotos).

  • git init --template=<directorio>arrow_right

    Permite inicializar el repositorio usando una plantilla de configuración específica.

  • git init --initial-branch=<nombre>arrow_right

    Establece el nombre de la rama principal al crear el repositorio.

  • git init --separate-git-dir=<ruta>arrow_right

    Guarda los metadatos del repositorio en otro directorio.

terminal Ejecutar

$ mkdir mi-proyecto 
$ cd mi-proyecto 
$ git init
